Generate Ethereum Wallet Keys

A fully offline, open-source, BIP44-compatible CLI tool for developers to generate ethereum wallet addresses and key-pairs from mnemonics. Simple, fast, and secure.

Get Started

Features

Secure & Offline

Generate key-pairs offline, without exposing your mnemonics to the internet.

Lightning Fast

Built with Rust, a language known for its performance and safety.

CLI Only

Built for developers with a focus on command-line efficiency and automation.

BIP44 Compatible

Generate addresses and key-pairs compatible with BIP44 standards.

Clone the Repository

$ git clone https://github.com/Parthipan-Natkunam/Wallet-Keys.git

Build & Install

$ cargo install --path .

Use the CLI

$ wallet-keygen --mnemonic "your twelve or twenty four word mnemonic phrase"
